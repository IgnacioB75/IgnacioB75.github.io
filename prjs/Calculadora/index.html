Tutorial: Calculadora

Esta es una aplicación de Calculadora básica.
Utiliza operaciones aritméticas básicas (suma, resta, multiplicación y división) para ver los resultados que desee el usuario.

Funciona con teclas numéricas (1,2,3...0), incluso las del teclado numérico!

Presiona "Enter" o "=" para ver el resultado.

Presiona "C" o "Backspace" para borrar.
